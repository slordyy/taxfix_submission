# Reports

A _Python_ reports app that provides an interactive marimo notebook environment build with uv.

## Prerequisites

* Python >= 3.13, you can use for example [uv](https://docs.astral.sh/uv/guides/install-python/) to manage that
* [uv](https://docs.astral.sh/uv/getting-started/installation/)

## Installing dependencies

```bash
make install
```

## Checks

To run all checks:

```bash
make check
```

### Auto-formatting

```bash
make auto-format
```

### Linting

```bash
make lint
```

## Running locally

To run locally:

```bash
make run
```

This will give you a link the output which you can click to open the notebooks.
Open the user_report.py file for the part 2 solution.
